### 🛡️ 输入验证
- **白名单原则**: 对所有外部输入（API 参数、表单、URL 查询）进行严格的类型和格式验证。
- **清洗输出**: 在将数据渲染到 HTML 或 JSON 之前，必须进行转义处理，防止 XSS。

### 🔐 认证与授权
- **密码存储**: 严禁明文存储密码。必须使用强哈希算法 (如 `bcrypt`, `argon2`) 加盐存储。
- **密钥管理**: 严禁在代码中硬编码 API Key、Secret 或数据库密码。必须使用环境变量。

### 🚫 常见漏洞防御
- **SQL 注入**: 始终使用 ORM 或预编译语句。
- **CSRF**: 状态变更请求必须验证 CSRF Token。
- **敏感数据**: 日志中严禁打印密码、Token 或 PII (个人身份信息)。
